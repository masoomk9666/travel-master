⚡ 30-SECOND CUSTOMIZATION
================================================================================

Make it Your Own:

1. Change Your Company Name
   File: components/Navbar.jsx (line ~30)
   Change: "Travel Master" → Your company name

2. Change Brand Colors
   File: app/globals.css (lines 11-14)
   Edit the color values to your brand colors

3. Update Contact Info
   File: app/contact/page.jsx (lines ~8-22)
   Update phone, email, address

That's the minimum! Your site now has your branding.

================================================================================
📚 COMMON TASKS
================================================================================

ADD A NEW DESTINATION
→ Open: app/destinations/page.jsx
→ Find: allDestinations array
→ Add: New destination object with name, image, price, etc.
→ Result: Automatically appears in the destinations page

CHANGE BRAND COLOR
→ Open: app/globals.css
→ Find: --primary: 3 105 161;
→ Change: The RGB numbers to your color
→ Result: Entire site updates instantly

ADD YOUR LOGO
→ Replace the ✈ emoji in components/Navbar.jsx
→ Or upload image and reference it
→ Result: Your logo appears in navigation

UPDATE ABOUT PAGE
→ Open: app/about/page.jsx
→ Edit: Company story, mission, team members
→ Result: Changes appear on About page

🚀 DEPLOY TO THE WORLD
================================================================================

Ready to launch? Follow these steps:

1. Push to GitHub
   → Create GitHub account (free at github.com)
   → Push your code to a repository

2. Deploy to Vercel (Recommended)
   → Go to vercel.com
   → Click "New Project"
   → Select your GitHub repository
   → Click "Deploy"
   → Your site is LIVE! 🎉

That's it! No complicated setup needed.

For detailed deployment instructions, see: DEPLOYMENT.md
